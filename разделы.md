## 🎯 Концепция бренда SITECLEAN

**Позиционирование:** "Мы делаем сайты чистыми — безопасными, быстрыми, работающими"

**Слоган:** *"SiteClean — чистый код, чистая репутация, чистый бизнес"*

**Ключевое преимущество:** Системный подход (не просто удаляем вирусы, а делаем сайт "чистым" во всех смыслах)

## 🚀 План запуска (приоритетность)

### **Этап 1: MVP (1-2 недели)**

**Что делаем сразу:**
1. ✅ Главная страница (все блоки)
2. ✅ Раздел "Услуги" → только /services/cleaning/ (общая страница)
3. ✅ Раздел "Цены" (3 тарифа)
4. ✅ Раздел "Контакты"
5. ✅ Раздел "Для бизнеса" (базовая информация)
6. ✅ Блог → 3-5 базовых статей

**Цель:** Начать принимать заявки и тестировать конверсию

---

### **Этап 2: Расширение (3-4 недели)**

**Что добавляем:**
1. ✅ Посадочные страницы под каждую CMS (WordPress, Joomla, Bitrix и т.д.)
2. ✅ Раздел "Кейсы" → 10-15 кейсов
3. ✅ Блог → еще 10-15 статей
4. ✅ Калькулятор стоимости
5. ✅ Раздел "Дополнительные услуги"

**Цель:** Увеличить органический трафик + конверсию

---

### **Этап 3: Масштабирование (2-3 месяца)**

**Что добавляем:**
1. ✅ Раздел "Абонентское обслуживание"
2. ✅ Раздел "Профилактика" (аудит, файрвол, мониторинг)
3. ✅ Раздел "Восстановление" (снятие блокировок)
4. ✅ Блог → 30-50 статей
5. ✅ Кейсы → 30-50 кейсов
6. ✅ Онлайн-сканер сайта (лид-магнит)
7. ✅ Telegram-бот для диагностики

**Цель:** Полноценный контент-сайт + дополнительные источники трафика
